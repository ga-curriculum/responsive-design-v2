# Responsive Design - Implementation Exampls

![Hero image](./assets/hero.png)

**Learning objective:** By the end of this lesson, students will recognize examples of responsive design. 




Here are some examples of how responsive web design can be used to improve the user experience:

On a desktop computer, a responsive web page might have a two-column layout with a sidebar on the left and the main content area on the right. On a mobile device, the same web page might have a single-column layout with the sidebar content below the main content area.
On a desktop computer, a responsive web page might have a large navigation bar with many different links. On a mobile device, the same web page might have a smaller navigation bar with a hamburger menu that can be expanded to reveal all of the links.
On a desktop computer, a responsive web page might have large images. On a mobile device, the same web page might use smaller images or adaptive images that resize to fit the screen size.
